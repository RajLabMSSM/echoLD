# Get lead SNP block

Identify the LD block in which the lead SNP resides.

## Usage

``` r
get_leadsnp_block(query_dat, ss, pct = 0.15, verbose = TRUE)
```

## Source

[adjclust GitHub](https://github.com/pneuvial/adjclust)

## Arguments

- pct:

  minimum percentage of points for the plateau selection in capushe
  selection. See [`DDSE`](https://rdrr.io/pkg/capushe/man/DDSE.html) for
  further details

- verbose:

  Print messages.
