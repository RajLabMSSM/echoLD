# Read LD list

Import existing LD results.

## Usage

``` r
read_LD_list(LD_path, query_dat, as_sparse = TRUE, verbose = TRUE)
```

## Arguments

- LD_path:

  Path to saved LD matrix.

- query_dat:

  Query data.frame.

- as_sparse:

  Convert the LD matrix to a sparse matrix.

- verbose:

  Print messages.

## Value

LD list with 3 elements.
